Política de Privacidade do DislexiApp

Temos o total compromisso de ser transparente com você. Confira abaixo com detalhes tudo o que você precisa saber sobre a segurança dos dados dos usuários que utilizam o DislexiApp.

DislexiApp é um aplicativo para crianças, com o objetivo de disponibilizar tarefas na forma de jogos atrativos e divertidos, ao mesmo tempo que trabalha o reconhecimento de figuras e a leitura de palavras.

Esta página é usada para informar os usuários sobre as políticas com a coleta, uso e divulgação de informações pessoais.

Ao acessar o aplicativo, você estará concordando e declarando estar ciente com o presente termo.

Utilização e Captação das Informações

1. PRIVACIDADE E DIREITOS DO USUÁRIO

1.1 Nome – necessário para identificar, bem como individualizar a experiência dentro do aplicativo.

1.2 Avatar – utilizado para personalizar a experiência do usuário. Este avatar é escolhido em uma lista pré-estabelecida pelo aplicativo, não é feito upload de imagem.

1.3 Quando quiser o usuário, a atualização dos dados cadastrais.

1.4 Se solicitado pelo usuário, a exclusão da conta a qualquer momento. Neste caso, o aplicativo não armazenará os dados pessoais que possui sobre o usuário.

2. PERMISSÕES DE DISPOSITIVOS

2.1 É necessário a permissão do microfone para que o aplicativo consiga realizar o reconhecimento de palavras.

2.2 A revogação de tal permissão poderá afetar o funcionamento apropriado desta plataforma.

3. PRESTADORES DE SERVIÇOS E ARMAZENAMENTO DE DADOS

3.1 São utilizados dois prestadores de serviços que fornecem o necessário para o armazenamento dos dados e funcionamento do aplicativo. São eles as empresas DigitalOcean e Amazon.

3.1.1 Dados do usuário que são armazenados pelas referidas empresas são: nome, senha, pontuação de perfil, nível do perfil, configurações, e avatar.

3.2 O áudio captado é enviado para o serviço de reconhecimento de voz Google Speech to Text, que não armazena gravações.

3.3 Os dados pessoais dos usuários não são vendidos, alugados ou divulgados para terceiros.

4. ALTERAÇÕES A ESTA POLÍTICA DE PRIVACIDADE

4.1 Este aplicativo reserva o direito de modificar a política de Privacidade a qualquer momento, bem como atualizar em decorrência de eventual alteração normativa, surtindo efeito imediato, razão pela qual se convida o usuário a consultar periodicamente esta seção.

4.2 Quando realizadas alterações os usuários serão notificados. Ao utilizar o serviço ou fornecer informações pessoais após eventuais modificações, o usuário demonstra sua concordância com as novas normas.

5. JURISDIÇÃO PARA RESOLUÇÃO DE CONFLITOS

5.1 Para a solução de controvérsias decorrentes do presente instrumento será aplicado integralmente o Direito brasileiro.

5.2 O presente documento foi elaborado em conformidade com a Lei Geral de Proteção de Dados (Lei 13.709/18) e o Marco Civil da Internet (Lei 12.965/14).
